# GOHSY FASHION TV — 4채널 방송국 기획서

## 1. 핵심 컨셉: Fashion MTV

MTV가 음악으로 패션 트렌드를 만들었다면, GOHSY는 패션으로 음악을 분위기 장치로 쓴다.

```
MTV:    음악(콘텐츠) → 뮤비(채널) → 패션 트렌드(부산물) → 소비
GOHSY:  패션(콘텐츠) → 4채널(스토리/소재/코스튬/교양) → 음악은 분위기 → 구매
```

### 핵심 차별점

```
일반 쇼핑몰:  눈(사진) → 머리(비교) → 지갑
GOHSY:       눈(영상) + 귀(음악) → 감정 → 지갑
```

머리를 건너뛴다. 비교 안 하고 느끼고 산다.

## 2. 아키텍처: 콘텐츠 채널 × 배포 채널

```
콘텐츠 채널 (무엇을 보여줄 것인가)
  CH1 STORY / CH2 MATERIAL / CH3 COSTUME / CH4 ACADEMY
        │
        ▼
배포 채널 (어디에 뿌릴 것인가)
  쿠팡 / 네이버 / YouTube / gohsy.com
```

4 × 4 매트릭스. 서로 충돌 안 한다. 쌓이는 구조.

## 3. 4 콘텐츠 채널

| # | 채널 | 드라이버 | 설명 | 예시 |
|---|------|---------|------|------|
| CH1 | **STORY** | 스토리 | Take On Me 컨셉 — 웹툰 → 실사 전환 → 구매 | "비 오는 날 입은 니트" 3컷 → 착용샷 → 구매 |
| CH2 | **MATERIAL** | 소재/장르 | 소재가 드라이버, 바로 구매로 연결. 짧다 | "린넨이란 무엇인가" → 린넨 셔츠 추천 → 구매 |
| CH3 | **COSTUME** | 캐릭터 | 영화 의상 설정처럼 역산 코디, 풀세트 판매 | "도쿄 프리랜서 룩" 풀세트 → 중국산 가성비 구성 → 구매 |
| CH4 | **ACADEMY** | 교육 | 80년대 일본 잡지 + 패션 교양 + 튜토리얼 | 패션학 강의 → 안목 형성 → 스스로 아이템 선택 |

### CH2 vs CH4 차이

```
CH2 MATERIAL: "이 소재가 좋다 → 이 제품 사라"  (소재 → 상품)
CH4 ACADEMY:  "패션이란 이런 거다 → 너가 골라봐라"  (교양 → 안목)
```

- CH2는 소재가 드라이버고 바로 구매로 간다. 짧다.
- CH4는 패션학 강의다. 시청자가 안목을 키운 다음 스스로 아이템을 고르게 만든다.
- **CH4는 팔지 않는다. 가르친다. 그런데 결국 산다.**

## 4. Take On Me 컨셉 — CH1 STORY 상세

### 영감

a-ha "Take On Me" 뮤직비디오 — 만화 세계와 실사 세계의 전환.

### 랜딩 페이지 구조

```
[HERO]
  전체 화면 웹툰 한 컷 — 캐릭터가 거리를 걷고 있다
  스크롤 유도 화살표

[EPISODE 1: PROD-001 니트 가디건]
  웹툰 3~4컷 (세로 스크롤)
  ├── 컷1: 캐릭터가 옷장 앞에서 고민
  ├── 컷2: 니트 가디건을 집어든다
  ├── 컷3: 거울 앞에서 착용 — 배경이 판타지로 전환
  └── 컷4: 클릭 유도 ("이 옷이 궁금하다면")
       ↓ 클릭 or 스크롤
  [REVEAL] ← ★ Take On Me 전환
  실사 모델 착용샷 (풀스크린, 프라다급 여백)
  가격 + 구매 버튼 (쿠팡/네이버)

[EPISODE 2: PROD-002 카고 팬츠]
  같은 구조 반복

[FOOTER]
  YouTube 채널 링크
  채널 아이콘 (쿠팡/네이버/YouTube)
```

### 전환 효과

CSS `mix-blend-mode` + 스크롤 기반 opacity 전환으로 웹툰 그림이 서서히 실사로 바뀌는 느낌. JS 애니메이션 아니고 CSS만으로 가능.

## 5. 음악 전략

| 상황 | 방법 | 저작권 | 비용 |
|------|------|--------|------|
| gohsy.com 도메인 | YouTube iframe 임베드 | 합법 (YouTube embed API 정책) | 0원 |
| gohsy.com 도메인 | Spotify 임베드 | 합법 (Spotify embed 정책) | 0원 |
| YouTube 업로드 영상 | parksy-audio 자체 렌더링 음원 | 자체 소유 | 0원 |

도메인에 들어오면 음악이 깔린다. 클럽처럼. 체류시간이 올라가고, 체류시간이 올라가면 전환율이 올라간다.

## 6. 채널 서핑 UX

```
┌─────────────────────────────────────────────┐
│  ◄ CH2 MATERIAL          CH4 ACADEMY ►      │
│              ▼ CH1 STORY ▼                  │
│                                             │
│        [현재 채널 콘텐츠 풀스크린]             │
│                                             │
│        ← 스와이프로 채널 전환 →              │
│                                             │
├─────────────────────────────────────────────┤
│  ♫ Now Playing: [곡명]  │ 👕 PROD-001 [$39] [구매]  │
└─────────────────────────────────────────────┘
```

### 하단 바 (Now Playing Bar)

- 왼쪽: 지금 재생 중인 음악 (Spotify/YouTube 임베드)
- 오른쪽: 현재 보이는 상품 + 즉시 구매 버튼
- **어떤 채널을 보든 하단 바에서 바로 산다**

## 7. 1상품 4채널 다중 노출 (빈도 효과)

```
PROD-001 니트 가디건 — 하나의 상품이 4채널에 전부 나온다

CH1: "비 오는 날 그녀가 입은 니트"           (스토리)
CH2: "메리노 울 vs 캐시미어 차이"            (소재)
CH3: "카페 사장님 룩 3종 세트 중 하나"        (코스튬)
CH4: "니트의 역사: 어부의 작업복에서 패션까지"  (교양)
```

마케팅 빈도 효과(frequency effect): 3회 노출 이후 구매 전환율 급등. 4채널이면 자동 4회 노출.
